# Couchbase query workbanch

You can override the following environment parameters:

```
GUI_PORT=:8094
COUCHBASE_URL=http://127.0.0.1:8091
USER=
PASS=
```

To give it a try run:

```
docker run --rm --net=host wearemakery/couchbase-query-workbench:dp2
```
